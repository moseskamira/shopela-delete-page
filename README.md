# Shopela Account Deletion Page

A simple web page for **Shopela** users to request deletion of their account and associated data, hosted via GitHub Pages.

### 🌐 Live Demo
[View the deletion request page](https://moseskamira.github.io/shopela-delete-page/)

---

## Features

- Allows users to submit account deletion requests easily
- Hosted securely on GitHub Pages
- Ready to link from the Shopela app or website
- Simple, responsive design

---
